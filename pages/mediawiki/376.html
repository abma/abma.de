Ngrep Examples


show all UDP packets
ngrep "" udp
show all ICMP packets
ngrep "" icmp
show TCP or UDP port 53 packets
ngrep "" port 53
show TCP port 23 packets
ngrep "" tcp port 23
listen on eth1
ngrep -d eth1 ""
show Microsoft browsing traffic for NT domain MS HOME
ngrep "MS HOME" port 138
show mail traffic
ngrep -iq "rcpt to|mail from" tcp port 25
show POP3 traffic
ngrep "user" port 110
Microsoft Windows fill the ICMP data with the alphabet; is the pinged host running Windows?
ngrep -q "abcd" icmp
show user agents from http traffic
ngrep -iq "user-agent" tcp port 80
show version of FTP server
ngrep "220" port 21
show ssh traffic
ngrep "SSH" port 22
show all traffic expect ssh
ngrep -v "" port 22
