Ausgehende Verbindungen werden blockiert


Plötzlich kein zugriff mehr auf Mailserver Port 25 möglich?
-Danke an Mcafee! Durch ein update, wurde das "feature" eingebaut, dass ausgehende Verbindungen blockiert werden.
so wirds deaktiviert:
  • rechtsklick auf das tray-icon->VirusScan-Konsole
  • rechtsklick auf Zugriffsschutz -> Eigenschaften
  • häckchen von "Senden von E-Mails durch Massenmail-Würmer verhindern" entfernen.
  • ok klicken
  • fenster schliessen
