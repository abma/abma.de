Exim4 mit imap server(configure local deliverly to maildir)


local_delivery:
driver = appendfile
create_directory
directory = $home/Maildir
directory_mode = 0775
maildir_format
user = $local_part

Kopie von ausgehenden Emails an Emailadresse senden

in exim.conf ganz am anfang einfügen:
system_filter = /etc/exim/printerredirect.filter system_filter_user = mail
und dann in /etc/exim/printerredirect.filter
Code:
  1. Exim filter
if $sender_host_address is "192.168.1.223"
then
headers add "Disposition-Notification-To: mail@host.domain"
headers remove from
headers add "From: mail@host.domain"
unseen deliver "mail@host.domain"
endif

Exim welcome Message takes a long time

This is a network problem, dns/ident requests couldn't be answered. => Exim is waiting for a timeout.
exim.conf
rfc1413_query_timeout = 1s

befehle

nachrichten in queue auflisten
exim -bp
queue jetzt abarbeiten
exim -qff
Nachricht aus der queue entfernen:
exim -Mrm Versionsnummer ausgeben
exim -bV
