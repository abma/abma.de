gexifrotate is a tool for setting very fast the exif orientation tag on many jpegs. gexifrotate main screen for reading and writing the jpeg-exif tags it uses jpegexiforient which comes with the jpeg tools/libs. (http://www.ijg.org/) usage: WARNING: be careful and make a backup before using this tool, because it's not tested. It could damage/destroy/delete all your jpegs! start, select a folder with jpegs. then the tool loads the first image. there are 6 buttons to control the program: * the first (ok) skips the current image and loads the next image * the second rotates the image 90
