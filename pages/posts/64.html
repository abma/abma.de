title: FUSE - Das Dateisystem der Zukunft ?! author: abma date: 03-08-2006 {% extends "post.html" %} {% block body %} {% load markup %} {% filter markdown %} Für das Filesystem in Userspace (FUSE) gibt es viele interessante Module, um nur ein paar zu nennen:
  • CopyFS (Versionierung aller Dateien)
  • Captive (Schreibzugriff auf NTFS-Partitionen)
  • SSHFS (Über SSH ein Verzeichnis mounten)
  • AVFS (Archive direkt bearbeiten)
Sogar ein Modul für ZFS ist in Entwicklung. Diese Module sollten sich sogar beliebig miteinander kombinieren lassen, in dem hintereinander die einzelnen Treiber läd. Auch GLScupe basiert auf FUSE. Ansich sind alle Module sehr interessant und auch relativ einfach einzurichten, da nur ein User-Mode programm benötigt wird. Auf der FUSE-Homepage gibt es eine Liste vieler interessanter FUSE-Treiber. {% endfilter %} {% endblock %}
