Howto make a Ebuild

if you get this error:
!!! /usr/local/portage/fastchat does not seem to have a valid PORTDIR structure.
then your filename isn't correct!
use a sample
then
ebuild fastchat-0.16.9999.ebuild digest
See also:
http://www.gentoo.org/proj/en/devrel/handbook/handbook.xml?part=2&chap=1#doc_chap2

Ebuilds manuell installieren


ebuild fetch unpack compile [test] [preinst] install [postinst] qmerge
es kann auch ein einzelner befehl ausgeführt werden z.B.
ebuild fastchat-0.16.9999.ebuild install
um ein gerade geändertes Ebuild zu installieren muss man
ebuild fastchat-0.16.9999.ebuild digest fetch compile install qmerge
eingeben.
siehe auch
man ebuild
