title: Windows disassembliert? author: abma date: 24-01-2006 {% extends "post.html" %} {% block body %} {% load markup %} {% filter markdown %} In dem Reactos-Forum ist die Diskussion entbrannt, ob Windows disassembliert wurde und dieser Assembler Code in ReactOS eingeflossen ist. Ich hoffe dem ist nicht so, denn dies würde dem Projekt mehr schaden als nützen, da dies in einigen Ländern illegal ist, und dies das Projekt angreifbar für Microsoft machen würde. Deswegen läuft im Moment wohl die Entwicklung auch etwas langsamer. Ein Entwickler hat andscheinend aus diesem Grund auch das ReactOS-Projekt verlassen. Update 24.01, 20 Uhr: Auf der ReactOS-Homepage sind fast alle Dienste offline. Im Moment funktioniert das Forum, der SVN-Server und die Mailinglist-Archive nicht. Ob dies im Zusammenhang steht ist noch unklar. Update2 25.01, 22 Uhr: Es gibt nun eine Offizielle Meldung auf der Homepage von ReactOS. ReactOS stoppt die Entwicklung bis alles geklärt ist. Aus diesem Grund wurden auch alle vorher genannten Dienste abgeschaltet. Update3 26.01, 23 Uhr: Auch die großen Newsseiten haben heute Nachmittag davon berichtet, dass die Entwicklung vorrübergehend gestoppt wurde. ( derstandard.at, golem.de, heise.de ) Update4 27.01,12 Uhr: Das ReactOS Team wird erst weiterentwickeln, nach dem der Quellcode von illegal enstandenem Quellcode gesäubert wurde, "auch wenn dies Jahre dauert". {% endfilter %} {% endblock %}
