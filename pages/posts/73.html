Caps-Lock? author: abma date: 01-09-2006 {% extends "post.html" %} {% block body %} {% load markup %} {% filter markdown %} Wer hat sich diese Tasten auf der Tastatur ausgedacht? Die Tasten sind doch sowas von sinnlos, für ein Wakeup kann man jede andere Taste verwenden, für ein Sleep (Standby) kann man sich eine beliebige Tastenkombination anlegen oder sich z.B. über das Startmenü in Windows hangeln, und für die Power-Taste gilt das selbe. So kann es mal leicht passieren, dass man auf die Power Taste kommt, und der Rechner schaltet sich ungewollt aus. (eben erst passiert) Und da regt sich noch jemand über die Caps-lock Taste auf! Vorher sollten diese drei Tasten von jeglichen Tastaturen verbannt werden. Da ist doch eine Tastatur mit OLEDs interessanter, jeder Taste kann so eine beliebige Funktion zugewiesen werden. Das geht im Moment zwar auch, allerdings kann sich keiner auf die schnelle ~100 Tasten merken. Da ist die Caps-Lock taste doch sehr sinnvoll, diese Taste ermöglicht es mit einem Finger zu Tippen, ohne diese Taste kann man manche wichtigen Tastenkombinationen gar nicht erst drücken. Es gibt zwar Eingabehilfen die dies Ermöglichen, aber mit da diese Taste an jeder Tastatur vorhanden ist, benötigt man die Eingabehilfen vielleicht gar nicht erst. {% endfilter %} {% endblock %}
